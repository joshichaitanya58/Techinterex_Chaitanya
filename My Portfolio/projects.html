Project-1 : Personal Portfolio Website

This project is a fully responsive and user-friendly personal portfolio website, developed using HTML, CSS, and PHP. It serves as an online representation of my skills, experience, and web development projects. The website features multiple sections including Home, About Me, Skills, Projects, and Contact, each thoughtfully designed to provide a smooth and engaging user experience across all devices—whether it's a mobile phone, tablet, or desktop.
The main objective of this portfolio is to showcase my technical abilities, creativity, and commitment to clean, efficient code. It also acts as a central hub where potential clients, collaborators, or employers can learn more about my background, view the projects I’ve worked on, and get in touch with me easily. This project not only demonstrates my proficiency in front-end and basic back-end development but also reflects my passion for building meaningful digital experiences.
